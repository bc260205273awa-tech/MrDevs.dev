# 🚀 MrDevs.dev — Premium Digital Agency

MrDevs is a premium digital agency website crafting high-performance, visually stunning websites, mobile applications, and custom software for ambitious businesses.

## 🛠️ Technology Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS & Vanilla CSS
- **Icons**: Lucide React
- **Fonts**: Syne (Headers) & Manrope (Body text)
- **Language**: TypeScript

## 🚀 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Run the Development Server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the live site.

## 📈 SEO & Sitemap Features

- **Dynamic Sitemap**: Automatically generated sitemap at `/sitemap.xml` via `app/sitemap.ts`.
- **Crawler Rules**: Defined via `app/robots.ts` pointing search bots to the sitemap.
- **Advanced Metadata**: Dynamic title templates, high-quality open-graph tags, canonical links, and bot-indexing rules in `app/layout.tsx`.
